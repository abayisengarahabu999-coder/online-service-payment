# online-service-payment
A static website that guides users through selecting and understanding online hosting services. This project demonstrates frontend development, responsive design, and clear user guidance.
---
##  Project Purpose
- Help users choose the right hosting service
- Explain server sizes and pricing
- Guide users through the payment process (demonstration only, no real payments)
